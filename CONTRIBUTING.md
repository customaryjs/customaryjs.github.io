# Update `customary.mjs`

When an example is modified to use a brand new Customary feature,
`customary.mjs` must be updated to include that feature.

1. In the Customary codebase, run `npm run dist`
2. Copy `.dist/bundled/customary.mjs` to `customary.mjs` in this repository
